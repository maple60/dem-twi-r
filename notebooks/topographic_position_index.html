地形位置指数の計算

地形位置指数 (Topological Position Index, TPI) は、地形の局所的な位置を定量化するための指標です。 TPIは、対象となる地点の標高と、その周囲の標高の平均値との差を計算することで求められます。正の値はその地点が周囲よりも高いことを示し、負の値は低いことを示します。

\[ TPI = 標高 - 周辺セルの平均標高 \]

Rのterraパッケージを使用して、DEM (Digital Elevation Model) からTPIを計算してみます。

データは、whiteboxパッケージのサンプルデータを使用します。

# install.packages("terra") # まだインストールしていない場合
# renv::install("terra") # renvを使用している場合
# install.packages("whitebox") # まだインストールしていない場合
# renv::install("whitebox") # renvを使用している場合
library(terra)
terra 1.9.34
library(whitebox)

dem_path <- sample_dem_data()
dem <- rast(dem_path)
plot(dem)

ある標高に対して、どの程度の周辺セルを参照するかを決めたうえで、terrain::focal()関数を使用して、周辺セルの平均標高を計算します。ここでは、3x3のウィンドウを使用して周辺セルの平均を計算します。

w <- matrix(1, nrow = 3, ncol = 3) # 3x3のウィンドウ
w[2, 2] <- NA # 中心セルは周辺平均から除外

mean_neigh <- focal(
  dem,
  w = w,
  fun = mean,
  na.rm = TRUE
)

tpi_3_3 <- dem - mean_neigh
plot(tpi_3_3, main = "TPI (3x3)")

解釈は以下のようになります。

TPI 意味
正 周囲より高い。尾根・凸地形寄り
0付近 周囲と同程度
負 周囲より低い。谷・凹地形寄り

どの規模でTPIを計算するかは、解析の目的に応じて決定する必要がありそうです。
